# NYC Taxi Analysis Streamlit App

Aplikasi ini dibuat untuk memenuhi tugas **Portofolio Building with Streamlit**. 

### Fitur Utama:
- Filter interaktif berdasarkan jarak perjalanan.
- Filter spesifik lokasi penjemputan.
- Visualisasi dinamis menggunakan Plotly.
- Ringkasan statistik (Metrics) otomatis.

### Cara Menjalankan Lokal:
1. `pip install -r requirements.txt`
2. `streamlit run app.py`
